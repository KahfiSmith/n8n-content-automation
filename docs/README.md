Fokus dokumentasi:

- struktur folder yang stabil

Dokumen yang penting untuk mapping perubahan:
- `02-workflows.md` untuk workflow aktif 1-2-3 dan future optional
- `06-repo-structure.md` untuk struktur aktual repo
- `07-handoff-manifest.md` untuk kontrak handoff
- `08-structure-mapping.md` untuk peta fungsi folder dan aturan perubahan
