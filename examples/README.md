# Examples

Folder ini berisi contoh kontrak data untuk integrasi Python clipper dan n8n.

Gunakan file di sini sebagai referensi:
- bentuk `manifest.json`
- payload hasil tiap stage
- `clip_caption_pack` untuk memastikan caption per clip sesuai jumlah video
- workflow aktif yang sudah dirapikan

Workflow aktif yang dipakai sebagai referensi:
- `wf-01-intake-clip-validate-assets.json`
- `wf-02-generate-caption-auto-schedule.json`
- `wf-03-publish-youtube-shorts-auto-schedule.json`
